struct = s3sum(filename[,blocksize])

top

Returns a struct bearing a checksum (.sum) and an error code (.err) for comparison to an S3 ETag field.

Blocksize specifies the size in bytes of multi-part upload chunks.

When blocksize is 0 then auto-select blocksize and upload checksum type.

When blocksize is -1 then treat as a single-part upload.

Error codes: 0 okay, 1 single-part warning, 2 file error, 3 checksum error
