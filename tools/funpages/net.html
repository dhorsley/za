call_details = web_serve_decode(webcallstruct)

top

Returns a struct representing details of an inbound web request.

web_serve_decode() returns the following member fields:

.host (host:port), .method, .path, .remote_ip (remote_ip:port) and .data (POST data).

Example:

define dynamic_request(call)

    c=web_serve_decode(call)

    h=c.host
    p=c.path
    r=c.remote
    m=c.method

    # ... rest of request handler

enddef
