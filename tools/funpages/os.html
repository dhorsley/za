array = dir([filepath[,filter]])

top

Returns an array containing file information on path filepath. filter can be specified, as a regex, to narrow results. Each array element contains name,mode,size,mtime and isdir key-value pairs. These specify filename, file mode, file size, modification time and directory status respectively.
